Script de Cumpleaños para More 🎂

Código de diseño corregido y optimizado usando Python Turtle

✨ Instrucciones de Uso

  • Asegúrate de tener instalado Python en tu sistema operativo.
  • Copia todo el contenido del bloque de código oscuro de abajo.
  • Guárdalo en un archivo de texto con el nombre feliz_cumple_more.py.
  • Ábrelo desde tu terminal o editor de código (como VS Code) y dale a ejecutar. ¡El pastel se dibujará automáticamente!

💻 Código en Python

import turtle

# Configuración inicial de la pantalla y la tortuga
t = turtle.Turtle()
s = turtle.Screen()
s.setup(width=800, height=600)
s.bgcolor("#FFF9E6")  # Fondo amarillo pastel suave
t.pensize(3)
t.speed(8)
t.hideturtle()

def dibujar_rectangulo(x, y, ancho, alto, color):
    t.penup()
    t.goto(x, y)
    t.pendown()
    t.color(color)
    t.begin_fill()
    for _ in range(2):
        t.forward(ancho)
        t.left(90)
        t.forward(alto)
        t.left(90)
    t.end_fill()

# --- 1. PRIMER PISO (Base - Marrón/Chocolate) ---
dibujar_rectangulo(-120, -50, 240, 60, "#5C2C16")

# --- 2. SEGUNDO PISO (Medio - Naranja/Zanahoria) ---
dibujar_rectangulo(-90, 10, 180, 50, "#E67E22")

# --- 3. TERCER PISO (Superior - Magenta/Fresa) ---
dibujar_rectangulo(-60, 60, 120, 50, "#E91E63")

# --- 4. LA VELA (Centrada en X=0) ---
t.penup()
t.goto(-5, 110)
t.pendown()
t.color("#F1C40F")
t.begin_fill()
for _ in range(2):
    t.forward(10)
    t.left(90)
    t.forward(30)
    t.left(90)
t.end_fill()

# --- 5. LA LLAMA DE LA VELA ---
t.penup()
t.goto(0, 145)
t.pendown()
t.color("#FF5722")
t.begin_fill()
t.circle(7)
t.end_fill()

# --- 6. TEXTO DE FELICITACIÓN ---
t.penup()
t.color("#2C3E50")
t.goto(0, -120)
t.write("¡Feliz Cumpleaños, More! 🎂", align="center", font=("Arial", 24, "bold"))

# Mantener la ventana abierta
turtle.done()
